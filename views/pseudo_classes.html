Explicando

As pseudo-classes servem para definir o estado de um elemento. Não é exatamente uma lógica de programação, mas é uma forma de o CSS entender o que está acontecendo no momento. Por exemplo: se o usuário passa o mouse em cima de um link, se ele clica em um botão ou se ele já visitou aquela página. É como se a gente desse uma instrução para o estilo mudar dependendo da interação do usuário.
