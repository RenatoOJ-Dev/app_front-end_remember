Box Model.

Tudo no CSS tem um quadro em volta de si. Os CSS são como caixas, box ou block, e tudo gira em torno disso.

Para poder moldar todas essas caixas ou blocos, precisamos de propriedades da qual podemos são usadas para dar tamanho a esses quadros.

Por exemplo, vamos usar uma <div> para demonstrar o comportamento de um quadro dentro do HTML e CSS.
